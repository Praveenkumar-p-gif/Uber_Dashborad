# üöñ Uber Analytics Dashboard (Power BI)

This project provides an interactive **Uber Ride Analysis Dashboard** built using **Power BI**.  
The dashboard helps analyze key metrics such as **Bookings**, **Revenue**, **Customer Behavior**, **Rider Trends**, **Distance Insights**, and **Vehicle Performance**.

---

## üìä Key Features

| Feature | Description |
|--------|-------------|
| **Booking Overview** | Total completed, lost, and uncompleted bookings with monthly trends |
| **Revenue Analysis** | Revenue breakdown by vehicle type, payment method, and customer segment |
| **Vehicle Performance** | Customer count, completed rides, and revenue comparison across vehicle types |
| **Rider Analysis** | First-time, returning, and regular rider distribution |
| **Location Insights** | Pickup hotspots, time-slot wise ride count, and distance covered |
| **Interactive Filters** | Vehicle category selection and monthly/quarterly toggle options |

---

## üìÅ Pages in Dashboard

| Page Name | Purpose |
|---------|---------|
| **Overview** | High-level performance summary of Uber rides |
| **Vehicle** | Revenue and booking comparison across Auto, Bike, Go Mini, Sedan, etc. |
| **Revenue** | Payment-based revenue and customer-wise contribution |
| **Rider** | Rider category analysis and customer behavioral patterns |
| **Location** | Location-based demand and time-split ride density |
